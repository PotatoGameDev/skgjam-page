How to create a jam submission page and upload my jam project?

The basic way is:

  1. Go to the jam page, click "Submit your project".
  2. Click "Upload game".
  3. On the "Create a new project" form you fill in all the data and save.
  4. !!! After saving, select Visibility & access - Public, this is crucial! Then save again.
  5. Go back to the jam page and click "Submit your project".
  6. Select the project you created in steps 3-4 in "Existing project" dropdown.
  7. Click "Submit"

How to prepare a Web build for my submission?

The current edition of the jam requires you to upload the game in web format. This is for security reasons. Your game WILL be brutally disqualified if it lacks a web version!
